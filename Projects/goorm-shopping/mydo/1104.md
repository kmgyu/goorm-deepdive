결제 완료
일단 결제 완료할 때랑 쿠폰 사용할 때의 메서드를 다른 것을 써줘야 할 것으로 보임.
결제가 완료되면 쿠폰도 같이 사용을 했다라는 메서드를 주면 좋을 듯.

중간에 트랜잭션 실패하면 얘네가 구분안해줘서 롤백할 수도 있나 걱정해봤는데 이렇게됨.
1. A에서 B 호출
2. B 완료 
3. A에서 B의 다른 메서드 호출 
4. B 실패
일단 gpt 말로는 이때 서비스의 트랜잭션 전부 롤백한다고 함.
