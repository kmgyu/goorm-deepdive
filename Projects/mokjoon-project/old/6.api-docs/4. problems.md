### Problems

1. 문제 검색  
    GET /api/v1/problems  
    query: q, tag, difficultyMin, difficultyMax, page, size, sort=difficulty,asc
    

```
200
{
  "content":[
    { "problemId":1000,"title":"A+B","difficulty":1,"tags":["implementation"],"acceptedCount":123456 }
  ],
  "page":0,"size":20,"totalElements":23000,"totalPages":1150
}
```
