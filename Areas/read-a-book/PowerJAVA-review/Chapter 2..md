비트 시프트 연산
`>>>`는 unsigned. 왼쪽 비트가 부호비트로 채워지지 않는다.
`>>`는 일반 시프트 연산.

```
-2 >> 2 하기

1073741822 // >>> : 00111111 11111111 11111111 11111110
```
