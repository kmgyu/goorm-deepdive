### Communicate using Markdown
마크다운 파일을 이용해 문서 작성하는 법을 배우는 파트
1. 헤더 생성
2. 이미지 삽입
3. 코드 스니펫 예제
4. 작업 리스트 만들어보기
5. PR 머지
다섯 가지를 실습한다.

위와 동일하게... 모두 해본 내용이기에 실습은 생략하였음...

공식 문서니까 당연하지만... 레포트로 활용할 수 있을 정도로 양질의 정보였다.
```

# 야매로 md 문법을 배웠는데, 처음 안 사실이어서 흥미로웠다.
```
