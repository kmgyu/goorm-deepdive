SonarQube for IDE?
실시간 코드 품질 분석 플러그인
인텔리제이, 이클립스, vsc 등에서 사용 가능.

버그, 취약점, 코드 스멜을 체크해준다.
- **Bug (버그)**: NullPointerException 가능성
- **Vulnerability (취약점)**: 보안 위험 코드
- **Code Smell (코드 냄새)**: 가독성·유지보수성 저하 코드
